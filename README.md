koa-starter
===========

undoZen's pick on starting a koa project

Installation
------------

1. install node v0.11.12+, [nvm](https://github.com/creationix/nvm) recommended: `nvm install v0.11.12`
2. `npm install`
3. `npm install -g bower`
4. `bower install` in client/
5. `npm install -g gulp`
6. `gulp`
